# 🏥 Health Record System

A Spring Boot-based Health Record Management System that enables secure storage and access of health records, user data, and related services.

## ✅ Features

- 🔐 JWT-based Authentication & Authorization
- 📧 Email Sending Feature (e.g., for registration or notifications)
- 📂 File Upload & Storage Handling
- 🔎 Search & Filter Capabilities for records/users
- 🛢️ MySQL Database Integration
- 📁 RESTful APIs for operations
- 🧑‍💻 Admin and User Management Modules

## 🛠 Tech Stack

- Java 17+
- Spring Boot
- Spring Security (with JWT)
- Spring Data JPA
- MySQL
- JavaMailSender
- Maven
